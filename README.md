# VideoDownloaderPro

A desktop application for downloading and viewing videos from various platforms like YouTube, TikTok, Instagram, and X.

## Features

- Simple and intuitive UI.
- Support for multiple video platforms.
- High-quality downloads.

## Prerequisites

For optimal video extraction performance, it is highly recommended to have a JavaScript runtime installed on your system:
- **[Deno](https://deno.land/)** (Recommended)

Without a JS runtime, `yt-dlp` may not be able to extract all available formats from some platforms.
